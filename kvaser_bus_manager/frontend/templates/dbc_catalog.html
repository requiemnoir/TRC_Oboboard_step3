Catalogo DBC (persistente)
Come funziona
  • Questa pagina legge i file .dbc in databases/dbc e li importa in un DB SQLite per ricerche veloci.
  • Le descrizioni arrivano dai commenti nel DBC (se presenti). Spesso sono sui segnali più che sui messaggi.
  • Se aggiungi un nuovo DBC nella cartella, clicca Importa tutti (oppure importa solo quello selezionato).
Se non lo vedi: assicurati che il file sia in kvaser_bus_manager/databases/dbc.
Più alto = più lento.
"Carica catalogo" usa il DB persistente. "Anteprima LIVE" usa il parser DBC e può essere più lenta.
